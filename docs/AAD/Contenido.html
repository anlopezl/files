Resumen del curso Course overview

Este curso presenta modelos estadísticos avanzados para el análisis de datos, sus fundamentos teóricos y aplicaciones en el software R. El curso cubre generalizaciones al modelo de regresión lineal que permiten mayor flexibilidad de estimación y técnicas de evaluación de modelos. También se introducen modelos de datos estructurados y variables latentes.

This course presents advanced statistical models for data analysis, their theoretical foundations, and applications using R software. The course covers generalizations to the linear regression model that allow greater estimation flexibility and model evaluation techniques. Structured data models and latent variables are also introduced.
